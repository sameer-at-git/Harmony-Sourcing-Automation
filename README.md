# Harmony-Sourcing Automation using Selenium

Automated using Selenium and Python with modular coding for resuability and reproducibility .

## Setup
- clone the repo 
- create venv for safer practice
- install pacjages form requirements.txt
- match google chrome version and download stable chromedriver
- replace the one from the repo if needed
- to start from scratch execute template.py once and the whole folder strucutre will be created, else clone repor and good to go 
- run main.py and select desired options
